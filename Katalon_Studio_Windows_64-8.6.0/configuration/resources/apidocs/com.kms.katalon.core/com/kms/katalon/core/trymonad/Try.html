Package: com.kms.katalon.core.trymonad

[Java] Class Try<T>

    • com.kms.katalon.core.trymonad.Try

    • Monadic Try type. Represents a result type that could have succeeded with type T or failed with a Throwable. Originally was Exception but due to seeing issues with eg play with checked Throwable, And also seeing that Scala deals with throwable, I made the decision to change it to use Throwable.
